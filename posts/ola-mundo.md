Este é o início de uma nova abordagem para a distribuição de conteúdo. Ao ler este texto, o teu navegador está a renderizar **Markdown** em tempo real, transformando dados brutos em informação visual.

## Porquê Markdown?
O Markdown é a linguagem franca da documentação técnica. É limpo, portátil e agnóstico de plataforma.

* Simplicidade
* Portabilidade
* Código Aberto
